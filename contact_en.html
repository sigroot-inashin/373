About the Website

Terms of Use

This website was created using a borrowed template.

The links and other content on this website are the personal property of the creator, but the template is a distributed one.
Please refrain from analyzing the website's data and creating an identical one.

Layout for each device

It is recommended to use a tablet or smartphone, but you can also perform a temporary check with the latest browser (IE10 or later for IE). If you narrow the browser's width, the layout will adjust to the size of each device.

Environment for small devices (width 900px or less)

The main menu becomes collapsible (a 3-bar icon).

Notes on this Website

Note: Please do not misuse the copyrighted works on the website (this includes the template and sites created by the developer).

As mentioned above regarding the template, please do not copy and paste URLs of sites on the website and claim them as your own creation. If you see this, please report it.

Regarding errors on the website

We make every effort to correct errors during creation, but if you find any, please contact the developer's DISCORD account.
Also, please contact us if you have any requests.

If a warning message appears during preview (for some browsers)

This warning is mainly caused by the javascript (js file) and does not appear on the web. Also, while this warning is displayed, you cannot see the effect, so please click inside the warning message to dismiss it. This will allow you to see the effect correctly.
